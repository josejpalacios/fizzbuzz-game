Rules

Select the number of CPU opponents to play against (Default 1).
Select the number to act as goal to count up too (Default 50).
Aim for your best score possible!

How to Play

Select Number if number is not divisible by 3 or 5.
Select Fizz if number is divisible by 3.
Select Buzz if number is divisible by 5.
Select FizzBuzz if number is divisible by 3 and 5.
